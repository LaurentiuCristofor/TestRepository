Formatting text is made easier by the Calibre editor because its toolbar provides quick access to bold, italics, underlined, and strikethrough.

You can include references within your ebook or to Web URLs using the Insert hyperlink action. For example, here's an internal reference to the Table of Contents and here's an external reference to Calibre's download URL: https://calibre-ebook.com/download.

Paragraphs are created using the <p> tag. If you need a line break within a paragraph, you can use the <br/> tag.
For example, this sentence follows a line break.

If you want to provide HTML or other code examples, you can use the <code> tag. To include the < and > characters under the tag <code>, you should use the escape sequences &lt; and &gt;, respectively.

Aligning paragraphs is also easy using the Calibre editor. For example, this paragraph was aligned to the right using the Align right action in the editor.

Sometimes you may want to format some data in a table. This can be done using the <table> tag along with the <tr>, <th>, and <td> tags, which stand for table header, table row, and table data, respectively. The content of the table will be defined row by row, with each row listing the header names or data values that compose it. Below is an example of a table; examining the HTML source of this chapter will show how the table tags are used to produce this result:

Table example
Name ID Birth Year
Alice 123456 2000
Bob 123457 2001

Note that this table example has a title; that title has been provided by using the <caption> tag. Also, the table borders are not present by default, but they are included here courtesy of our stylesheet.css style file, which specifies their presence as well as the centering of the table.
